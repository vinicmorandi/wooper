## Para iniciar o projeto

Crie, em seu banco de dados, uma database chamada `pokemon`

Vá para `src/API` e rode no cmd o seguinte comando:

#### `node index.js`

Após isso, volte à pasta inicial e rode o seguinte comando:
#### `npm start`
